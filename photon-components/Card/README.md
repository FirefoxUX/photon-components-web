# photon-cards

## Styles

### Default

```html
  <div class="card">
    ...
  </div>
```

```html
  <Card>
    ...
  </Card>
```

### Optional shadow

(`.shadow-10` defined in `attributes/shadows.css`)

```html
  <div class="card shadow-10">
    ...
  </div>
```

```html
  <Card shadow={true}>
    ...
  </Card>
```

## Sizes

### Large

```html
  <div class="card--size-large">
    ...
  </div>
```

```html
  <Card isLarge={true}>
    ...
  </Card>
```
